Hey, this is my new game using HTML, CSS, and JavaScript. At first, I was going to call it Number Detective, but it sounded weird, so I combined Numbers + Memory, which is what the game is about, and called it Numbery! I got a little inspired by Among Us with the whole impostor thing, haha. Well, this game only uses the space bar! You only need the mouse to change the difficulty when you lose. I think this is one of the best games I’ve made because I managed to make it in under 8 hours :D

You have 3 difficulties. The harder the difficulty, the more numbers there are. The trick here is that numbers keep appearing, and you have to let the new ones pass while duplicates (impostors) show up. You have to reject them by waiting 2-3 seconds since using the space bar means pressing space! 

Go ahead and try it, I don’t want to spoil it, haha!
